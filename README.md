# University Academic Analytics System

A NoSQL-based academic analytics system designed to track student performance, identify at-risk students, and analyze course difficulty using MongoDB aggregation pipelines.

---

## Features

- Student performance tracking system
- Course difficulty analysis
- Identification of at-risk students
- Flexible schema design using MongoDB documents
- Advanced aggregation queries
- Data-driven academic reporting
- Index optimization for query performance

---

## Tech Stack

- MongoDB
- MongoDB Aggregation Framework
- Node.js (if used) or script-based queries
- NoSQL document modeling

---

## Database Design

- Students collection
- Courses collection
- Enrollments collection
- Grades / Performance tracking collection

---

## Advanced Queries

- `$lookup` for relational-style joins
- `$facet` for multi-output analytics
- `$group` for statistical aggregation
- `$switch` for conditional logic
- Index optimization using `explain()`

---

## Highlights

- NoSQL flexible schema design
- Advanced aggregation pipelines
- Academic performance analytics system
- Scalable database architecture
